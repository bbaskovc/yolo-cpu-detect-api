# Model Artifacts

This directory is a documentation anchor only. Do **not** commit model weights, labels, generated model metadata, or derived artifacts here.

Approved deployment artifacts must be supplied outside Git and recorded through the model approval gate in `docs/13-licensing-and-model-policy.md`:

- ONNX weight path;
- model SHA-256;
- label-map SHA-256;
- model/license provenance;
- export process and tool versions;
- public model ID;
- preprocessing and threshold policy.
